🧠 Health Education AI

An intelligent AI-powered healthcare education platform that provides personalized, easy-to-understand medical information based on user input.

🚀 Features
🧠 AI-Powered Health Guidance
Get detailed explanations about diseases, symptoms, treatments, diet, and lifestyle.
💬 Chat Interface
Ask follow-up questions like a conversation with an AI assistant.
📊 Risk Dashboard
Visual risk level (Low 🟢 / Medium 🟡 / High 🔴) for better understanding.
📄 PDF Report Download
Generate and download a clean, structured health report.
🌍 Multi-language Support
Supports English, Tamil, and Hindi for better accessibility.
🔊 Voice Output
Listen to health information in your selected language.
🎤 Voice Input
Speak instead of typing (speech-to-text).
🖼️ Relevant Medical Visuals
Displays condition-related images for better understanding.
🛠️ Tech Stack
HTML
CSS
JavaScript
OpenRouter API (AI generation)
jsPDF (PDF generation)
Web Speech API (Voice input/output)
